Options Chain API Documentation

API Endpoints

GET /api/v1/options/{symbol}

Retrieve options chain data for a given stock symbol.

Parameters:

  • symbol (path) - Stock symbol (e.g., AAPL)
  • expiration (query) - Expiration date (YYYY-MM-DD)
  • min_strike (query) - Minimum strike price
  • max_strike (query) - Maximum strike price

Example Request:

curl -X GET "http://localhost:5000/api/v1/options/AAPL?expiration=2024-01-19&min_strike=150&max_strike=200"

Example Response:

{
    "symbol": "AAPL",
    "expiration": "2024-01-19",
    "calls": [...],
    "puts": [...]
}

Rate Limiting

Requests are limited to 100 per minute per IP address.

Error Codes

  • 400 - Bad Request (invalid parameters)
  • 404 - Data Not Found
  • 429 - Rate Limit Exceeded
  • 500 - Internal Server Error
